Case 1: Extra Sensory Perception

We now consider a second example described by Sprenger (2013).

Sprenger’s example: Jahn (1987) used electronic and quantum-mechanical random event generators with visual feedback; the subject with alleged psychokinetic ability tries to “influence” the generator. The number of “successes” in this experiment was \(s=52,263,470\) and the number of trials was \(n=104,490,000\).

Assuming independence of each trial we have \[ x_i|\rho \sim \mbox{Bernoulli}(\rho), \]

where \(1\le i\le n\) with \(p\in[0,1]\).

If we perform the test \[ H_0\colon \rho=0.5 \qquad \mbox{versus} \qquad H_1\colon \rho\ne 0.5, \]

then a rejection of \(H_0\) leads to evidence that the subject has psychokinetic ability. Using the data a classical hypothesis testing approach leads to a \(p\)-value approximately equal to \(0.0003\), leading to a rejection of the null hypothesis for the \(\alpha=0.05\) cut-off. A 95% confidence interval for \(\rho\) is \((0.50008, 0.50027)\).
